## WEAPON SAVE

### Option: Add new

- Function: add weapon_to_save

### Option: Equip

- Show when a similar weapon without owner is selected
- Show a checkbox for also updating this selected weapon if it's different from the weapon_to_save
- Function: connect the selected weapon and the character (+update the weapon)

### Option: Update

- Show when the current weapon is similar to the weapon_to_save and is selected
- Function: update the current weapon

### Option: Skip

- Show when there's current weapon
- Function: no action, keep using current weapon
