# Save feature

## Save individual item

- Similar Items not found -> Auto Save
- Similar Items found -> Select to Duplicate

## Save character build

Offer multiple-step saving options

### New character

1. Tempo save character
2. Weapon:

- Similar weapon not found -> Continue
- Similar no owner weapon found -> Select one to Equip | Add new

3. Artifact:

- Similar artifact not found -> Continue
- Similar artifact found -> Select one to Equip (no owner) or to Take (from owner) | Add new

### Old character

1. Compare character

- Same -> Continue
- Different -> Ask to Update

2. Weapon:

- Same weapon & same stats -> Continue
- Same weapon & different stats -> Ask to Update
- Different weapon -> Ask to Switch
  - If Switch, find similar no owner weapon -> Select to equip | Add new
  - If not Switch, keep current weapon -> Continue

3. Artifact:

- Same artifact & same stats -> Continue
- Same artifact & different stats -> Ask to Update
- Different artifact -> Ask to Switch
  - If Switch, find similar artifact -> artifact
  - If not Switch, keep current artifact -> Continue
